# Local prompt reference: Logic check

Source basis: the "逻辑检查" section of Leey21/awesome-ai-research-writing.

Use this reference when the task is a final consistency and logic review rather than open-ended rewriting.

Core rules to enforce:
- Assume the current draft has already gone through multiple rounds and is reasonably mature.
- Report only material problems: logic breaks, terminology drift, or serious grammar issues that obstruct meaning.
- Do not make ornamental edits just to sound helpful.
- Treat this as a red-line review, not as an excuse for style churn.

Output expectation for the subagent:
- if no substantive issue exists, say so briefly
- if substantive issues exist, report them concisely and concretely
