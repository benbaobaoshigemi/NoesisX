# Local prompt reference: Reviewer lens

Source basis: the "论文整体以 Reviewer 视角进行审视" section of Leey21/awesome-ai-research-writing.

Use this reference when the requested deliverable is a critical review, harsh evaluation, rejection-risk memo, or reviewer-style report.

Core rules to enforce:
- Start from a skeptical posture.
- Skip empty praise and move quickly to weaknesses that matter.
- Focus on originality, rigor, and consistency between claimed contributions and evidence.
- Prefer concise, concrete criticism over long atmospheric commentary.
- Keep the result constructive even when harsh.

Output expectation for the subagent:
- a reviewer-style report with clear strengths, critical weaknesses, and action-oriented advice
