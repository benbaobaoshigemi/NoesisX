# Compliance checklist

Use this when the user did not request polishing.

Check:
- Does the draft match the requested deliverable type?
- Does it follow the selected local prompt references?
- Is the structure coherent and readable?
- Are there obvious AI-style residues that the selected references explicitly forbid?
- Are there logical inconsistencies or obvious terminology drifts?

If there is a serious mismatch, revise only to the extent needed to satisfy the selected references. Do not turn this into a full polishing pass unless the user asked for one.
