---
name: writer-chinese-de-ai
description: rewrite Chinese academic or report-style prose to reduce obvious machine-like residue while preserving meaning, evidence, uncertainty, and document scope.
---

Use this skill inside `writer` when the user explicitly wants a Chinese text to feel less machine-like.

Before writing, read [references/prompt.md](references/prompt.md).

Core rules:

- reduce stiffness and template-like transitions
- preserve meaning, evidence, and uncertainty
- avoid over-rewriting when the text is already natural
- do not invent new content

Output:

- return the cleaned Chinese text
- default to no extra commentary
