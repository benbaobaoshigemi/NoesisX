Use this skill inside `writer` when the document should include existing figures or charts from `visualization/`.

Before writing, read [references/prompt.md](references/prompt.md).

Core rules:

- use only real visualization assets
- place figures where they support the surrounding argument
- keep captions, numbering, and in-text references consistent
- do not treat images as decoration without explanatory value

Output:

- image integration plan or updated document figure placement guidance
