# Chinese Reviewer Lens

Source basis:

- <https://github.com/SYSUSELab/academic-writing-guide>
- local adaptation from the existing `reviewer-lens` reference

Use this seed when the requested deliverable is a reviewer-style critique, harsh internal review memo, rejection-risk assessment, or critical evaluation written in Chinese.

Core rules to enforce:

- 以审稿人视角出发，先看问题是否成立，再看表述是否漂亮。
- 快速进入关键问题，不写空洞表扬。
- 重点检查原创性、方法严谨性、实验或证据是否足以支撑宣称。
- 检查贡献表述是否夸大，是否存在“标题很大、证据很弱”的失衡。
- 检查逻辑是否自洽，实验设置或比较对象是否公平。
- 批评要具体、可执行，避免情绪化挖苦。
- 允许严厉，但不允许故作刻薄。

Output expectation:

- 给出 reviewer 风格的中文评议
- 可包含：主要优点、关键缺陷、风险等级、修改建议
- 重点应放在真正影响可接受性的事项上
