# discover examples

## example: information exhaustion boundary

A dataset, its logs, generated plots, and external references have all been re-checked. Re-reading only rephrases earlier observations. No meaningful new signal appears. Stopping is justified under information exhaustion.

## example: path exhaustion boundary

Several hypothesis cuts, alternative explanatory frames, and reasonable tests have already been run. An additional round would merely restate earlier tests under slightly different labels. Stopping is justified under path exhaustion.

## example: critical missing input boundary

The central variable definition required for interpretation is absent from the current repository and cannot be reconstructed from existing metadata. The issue is not lack of effort but lack of a decisive input.

## example: contamination-risk boundary

The remaining analysis would require leaning harder on a very small and noisy sample. Any additional claim would become more opportunistic than evidential. Stopping preserves research quality.

## example: no-net-new-signal after external perturbation boundary

The main agent has already explored the strongest available direction. A fresh one-shot subagent is asked to attack the line, look for blind spots, and search for overlooked branches. It returns no meaningful new path. Stopping is now justified.
