# Thesis Compliance Check

Source basis:

- <https://github.com/SYSUSELab/academic-writing-guide>
- <https://github.com/tuna/thuthesis>
- <https://github.com/JingWangTW/NYCU-Thesis-Template>
- <https://github.com/CSUcse/CSUthesis>

Use this seed when the deliverable is a Chinese thesis, dissertation chapter, graduation report, or school-template-oriented formal academic document.

Core rules to enforce:

- 优先检查结构完整性，再检查局部措辞。
- 检查摘要、目录、章节层次、图表、参考文献、附录等是否齐备。
- 检查中英文摘要、术语、图表编号、表题、参考文献格式是否前后一致。
- 检查章节标题风格是否统一，层次是否清晰。
- 检查图表引用、参考文献引用是否存在漏引、重引、错引。
- 若用户已经给出学校模板，以学校模板为最高规则；否则仅做通用学位论文规范检查。
- 不在未给模板的情况下伪造某校专属格式要求。
- 这是规范检查，不是全文重写。

Output expectation:

- 返回“合规问题清单”而不是重写全文
- 按严重度区分：必须修、建议修、可保持
