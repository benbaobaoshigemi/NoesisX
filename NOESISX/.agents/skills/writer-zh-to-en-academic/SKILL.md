---
name: writer-zh-to-en-academic
description: translate Chinese academic or technical prose into accurate academic English while preserving the original meaning, structure, and caution level.
---

Use this skill inside `writer` when the source text is Chinese and the target should be academic English.

Before writing, read [references/prompt.md](references/prompt.md).

Core rules:

- translate meaning, not just words
- preserve scope, uncertainty, and structure
- keep academic tone steady rather than flashy
- do not add missing claims by invention

Output:

- return the English translation
- keep formatting stable unless the user asked for reformatting
