# Injected Prompt Seed

Source basis:

- `../../push-writer/references/prompt-library/imported-prompts/bilingual-academic-translation.md`

Inject these rules into the writer brief:

- 先判断输入文本使用的是哪种语言。
- 再将其准确、学术化地翻译到目标语言。
- 不重复原文，除非任务明确要求双语对照。
- 使用稳定术语和有效写作经验，而不是机械逐词替换。
- 保留原文的谨慎程度、限定条件和论证边界。

Output expectation:

- 若需要，先点明源语言
- 再给出目标语言译文
