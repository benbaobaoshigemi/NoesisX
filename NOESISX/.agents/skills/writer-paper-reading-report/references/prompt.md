# Injected Prompt Seed

Source basis:

- `../../push-writer/references/prompt-library/imported-prompts/paper-reading-report.md`

Inject these rules into the writer brief:

- 先概述文献背景、研究目的和问题。
- 再梳理研究方法、关键数据、主要发现和结论。
- 对新颖概念做必要的通俗解释，帮助读者理解逻辑与创新点。
- 最后做优缺点评估，并指出可能的后续研究方向。
- 整体报告必须结构清晰、逻辑严谨。

Output expectation:

- 返回中文阅读报告
- 重点是“解释和评估”，不是原文摘抄
